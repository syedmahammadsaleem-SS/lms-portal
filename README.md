# 🎓 LMS Portal (AI Learning System)

🚀 This is a full Learning Management System built using Flask.

---

## 🌟 Features

- Login & Register
- Course Upload
- Video Learning
- File Upload
- Quiz System
- Certificate Download
- AI Chatbot

---

## 🛠 Technologies

- Python
- Flask
- SQLite
- HTML, CSS, Bootstrap

---

## 📂 Project Structure

lms-portal/
│
├── static/uploads
├── templates
├── app.py
├── database.db
├── requirements.txt

---

## ▶️ Run Project

Step 1:
pip install -r requirements.txt

Step 2:
python app.py

Step 3:
Open browser:
http://127.0.0.1:5000
